# Generate lajnz

Generate folders and files based on template files

## Getting Started

### Installing

`npm install @linusljung/generate-lajnz`

### How to use

- Create `.glzrc` in the project root

```json
{
  "types": {
    "package": {
      "rootDir": "src/packages"
    }
  }
}
```

- `npx glz init`
- `glz [add [package]]
